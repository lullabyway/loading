# loading
cute loading.

## 使用

在main.js中导入

import loading from 'loading'

Vue.use(loading)

在需要使用的页面使用 ``` <loading :isShow='loading'></loading>``` 标签

## API

|参数 | 说明 | 类型| 默认值|
|--| -- | --| -- |
|isShow | loading是否显示 | Boolean | true |
